Free Code Camp URL Shortener


When you pass a valid url to the end of https://ffc-urlshortener.herokuapp.com/ it will pass a JSON Object with the original url and a shorten url to be a shortcut

If it's a valid url string, when I visit the shorten url it will redirect me to my original link

However if the url is invalid, the JSON object will display text invalid URL


"newlocal" uses a JSON object to hold data. Data will be lost when the server turns off.

"new" uses a mongoDB to save data permanently


Example:

https://ffc-urlshortener.herokuapp.com/newlocal/https://www.google.com

https://ffc-urlshortener.herokuapp.com/new/https://www.google.com


{ "original": https://www.google.com, "shorten":https://ffc-urlshortener.herokuapp.com/getlocal0 }
